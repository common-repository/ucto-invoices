Plugin Účto+ invoices is designed for Wordpress (WooCommerce) online stores who have purchased invoicing software Účto+. Plugin automates the connecting and sending data from e-shop to invoicing system. You can create invoices directly from your e-shop orders.

== Description ==
For proper work of this plugin you need to have active subscription of Účto+.
With this plugin you can than create Účto+ invoices directly from your Woocommerce store.
Simply download the plugin, set up all necessary information including your Účto+ API Key and you can start creating your Účto+ invoices directly from e-shop Orders page.

Účto+ URL: https://uctoplus.sk
Účto+ Terms: https://uctoplus.sk/terms/

== Installation ==

This section describes how to install the plugin and get it working.

1. Download the plugin and upload files directly via FTP (`/wp-content/plugins/uctoplus).
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. In the main menu to the left you will see new menu item "Účto+ Invoices" which contains settings page.
4. Input all your settings to the plugin settings page.
5. After the plugin is set up, you can proceed to generate invoices from your orders.
6. Navigate to your orders screen, mark orders you wish to generate invoices for.
7. From the drop down menu (Bulk actions) select option to "Účto+ - generate invoice".
8. That's all folks!
